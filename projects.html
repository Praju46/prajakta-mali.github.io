Projects

Urban Flood Mapping using Sentinel-1 SAR (Mumbai)

This project focuses on delineating urban flood extents using multi-polarized Sentinel-1 SAR data. The study evaluates different polarization combinations and automated workflows for flood detection in dense urban environments.

Exploring Urban Water Balance using VIS Fraction Mapping

This M.Tech project analyzes urban water balance dynamics using Vegetation–Impervious–Soil (VIS) fraction mapping combined with hydrological modeling to understand runoff, infiltration, and evapotranspiration patterns.

Cauvery River Basin Management Studies

Contributed to integrated river basin management at IISc by generating multi-year flood inundation maps, geomorphological classifications, soil erosion estimates using RUSLE, and climate data analysis using CMIP6 datasets.

Google Earth Engine CMIP6 Data Explorer

Developed an interactive Google Earth Engine application to visualize CMIP6 climate model projections, enabling users to explore spatial and temporal climate trends.

GIS-Based Site Suitability Analysis for STPs

Conducted a GIS-based site suitability analysis for identifying optimal locations for sewage treatment plants using multi-criteria decision analysis techniques.
